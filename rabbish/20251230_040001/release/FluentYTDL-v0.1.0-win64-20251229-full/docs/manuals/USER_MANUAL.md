# FluentYTDL Pro 用户手册

欢迎使用 **FluentYTDL Pro**！这是一款基于 yt-dlp 引擎构建的现代化视频下载工具，旨在为您提供流畅、高效且美观的下载体验。

---

## 🚀 快速入门

### 1. 核心组件检查
首次使用前，请确保核心组件（**yt-dlp** 和 **FFmpeg**）已就绪。
- 程序启动时会自动检查组件更新。
- 您也可以在 **“设置 -> 核心组件”** 中查看版本状态，或手动点击 **“检查更新”**。
- 如果自动下载失败，可以使用 **“手动导入”** 按钮导入本地的 `.exe` 文件。

### 2. 下载视频
最简单的下载方式利用了**剪贴板监听**功能（需在设置中开启）：
1. **复制链接**：在浏览器中复制任意 YouTube 视频或播放列表链接。
2. **自动识别**：软件会自动检测并弹窗提示。
3. **确认下载**：点击弹窗中的 **“下载”** 按钮即可。

或者手动操作：
1. 点击左侧导航栏的 **“新建解析”**。
2. 粘贴链接并回车。
3. 等待解析完成后，选择格式并下载。

### 3. 下载管理
- 在 **“下载列表”** 页面，您可以查看所有任务的进度。
- **并发控制**：顶部工具栏可以调整同时下载的任务数量（建议设置为 3）。
- **批量操作**：点击 **“批量选择”** 按钮，可以对多个任务进行删除、暂停或开始。

## 🛠️ 进阶功能

### A+B 格式选择（专业模式）
FluentYTDL Pro 支持强大的流分离与组合功能：
1. 在解析成功后的弹窗（或播放列表中点击“选择格式”），您会看到格式选择器。
2. **简易模式**：直接选择预设（如“最佳画质 MP4”、“纯音频 MP3”）。
3. **专业模式**：切换到“专业模式”，您可以分别选择 **视频流**（如 4K 无声画面）和 **音频流**（如 128k 音频）。软件会自动将它们合并为一个完整的视频文件。

### Cookies 注入
下载会员视频或受限视频时，可能需要登录信息：
1. 在 **“设置 -> 核心组件”** 中，将 **Cookies 来源** 设为 **“从浏览器读取”**。
2. 选择您常用的浏览器（如 Chrome 或 Edge）。
3. 确保您已在该浏览器中登录 YouTube。
4. 再次尝试解析视频。

### 播放列表深度自定义
对于播放列表，您不必全盘接受默认画质：
- 点击列表中任意单项的 **“选择格式”** 按钮。
- 您可以为该特定视频指定特殊的格式组合（例如：只有这一集我要下载 4K，其他的 1080P 即可）。
- 列表按钮会显示 **“已自定义”** 状态。

## ❓ 常见问题 (FAQ)

**Q: 下载速度慢怎么办？**
A: 请检查您的网络连接。在 **“设置 -> 网络连接”** 中，尝试配置代理服务器（如 `127.0.0.1:7890`），这通常能显著提速。

**Q: 为什么提示“无法合并”？**
A: 确保 **FFmpeg** 已正确安装且被软件识别。没有 FFmpeg，软件无法将独立的视频流和音频流合并。

**Q: 软件打不开或闪退？**
A: 请尝试删除安装目录下的 `config.json` 文件重置配置，或查看 `logs/` 目录下的日志文件反馈给开发者。
